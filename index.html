Welcome to my Web site. I'm a writer, amongst various other things, and this is my old school HTML shopfront. That is definitely me on the left.

"But why?" you may ask, naive child that you are. Well, firstly, self-promotion. But also a desire to create a Web presence that doesn't rely on dodgy social media platforms.

I've also designed this page to be easy to read on the Lynx text browser, because Mozilla's on the skids, and a Web browser market dominated by Chromium is a bad idea for many reasons.

I would go so far as to say we strip away all the bloat and return to the original aims of the Web, but that's another rant.
